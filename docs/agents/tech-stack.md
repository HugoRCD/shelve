# Tech Stack

- Framework: Nuxt.
- UI: Nuxt UI, Tailwind CSS.
- Server: Nitro.
- Full-stack tooling: NuxtHub module (`@nuxthub/core`) with hub DB and hub KV.
- Database: PostgreSQL, DrizzleORM.
- Monorepo: Turborepo.
- Deployment: Vercel.
- Mailer: Resend.
- Validation: Zod.
- CI/CD: GitHub Actions.
